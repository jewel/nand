NAND Shell
====

The unix philosophy is that every program does one thing and does it well.  Taken to extreme, the only program we need is nand.

Installation
----

```sh
git clone https://github.com/jewel/nand.git
cd nand
```

Usage
-----

```sh
export PATH=.:$PATH
nand 0 0
# Should output "1"
```

```sh
2bit-adder 0 1 1 0

# Should output "1 1"
````
